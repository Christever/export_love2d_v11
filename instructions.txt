J’ai réalisé un script qui permet d’exporter votre jeu dans plusieurs formats depuis Windows:  PC Windows (32 bits, 64 bits), MAC OS X et WEB.

Lien We Transfert : https://we.tl/RZLRDJ7hxs

Télécharger l’archive et décompresser le tout dans un répertoire.



Ensuite placer vos fichiers de jeu lua  dans le dossier « Source », puis doubler cliquer ensuite sur le script build.bat

Vous trouverez ensuite dans leurs répertoires respectifs, BuildWeb, BuildDesktop (dossier versions 32 et 64 bit) et BuildMac vos jeux dans les bons formats.



Pour gagner du temps, le script vous fournit aussi dans chacun des répertoires une version compressée des fichiers,

qu’il vous reste plus qu’à mettre en ligne.
